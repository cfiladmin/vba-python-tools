# CLAUDE.md — Claude Code 副業ルールブック

## プロジェクト概要

Claude Codeを使ったクラウドソーシング副業。目標：月5万円（90日）

## 絶対ルール（AIコンプライアンス）

1. **AI使用を開示する** — 納品物・プロフィールに「AI使用・最終確認は人間が行います」と明記
2. **機密情報を入力しない** — 本番DB・クライアント個人情報・APIキーはClaude Codeへ直接入力禁止。ダミーデータで開発
3. **仮払い後に着手** — クラウドワークスは仮払い確認後に作業開始。例外なし
4. **禁止案件を断る勇気** — robots.txt違反スクレイピング・過剰負荷処理・法的グレー案件は即断る
5. **税金を管理する** — 副業年収20万円超は確定申告必須。Claude Pro代は経費計上可（青色申告）

## 狙うべき案件（優先順位）

| 優先度 | 分野 | 単価目安 |
|--------|------|----------|
| 最優先 | VBA・Excel自動化 | 5,000〜20,000円/件 |
| 最優先 | Pythonスクレイピング | 7,000〜30,000円/件 |
| 最優先 | プロンプト設計・運用 | 5,000〜10,000円/件 |
| 次点 | サイト軽微修正 | 3,000〜15,000円/件 |
| 次点 | 帳票・経理自動化 | 10,000〜50,000円/件 |
| 補助のみ | AI記事ライティング | 文字1〜2円（下落中） |

## 納品後の必須アクション

1. 納品直後に提案 →「月額メンテプラン 3,000円/月」
2. 1週間後にフォロー → 「類似ツールの追加作成はいかがでしょうか」
3. 評価依頼 → 納品完了時に丁寧にお願いする
