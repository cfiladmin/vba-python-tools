## 1. CSV整形・クレンジング

**→ 詳細: [Python/README.md](Python/README.md)**

クライアントから受け取った「汚いCSV」を自動でクリーンにします。

```
入力（7行）                    出力（4行）
  株式会社A , 田中  ,50000   →  株式会社A,田中太郎,50000
株式会社B,鈴木花子,30000      →  株式会社B,鈴木花子,30000
  株式会社A , 田中  ,50000   →  株式会社C,佐藤一郎,80000  ← 重複削除
，，，                        →  株式会社D,山田次郎,20000  ← 空行削除
```

## 2. Excel帳票自動生成

**→ 詳細: [Python/README.md](Python/README.md)**

Excelテンプレートにデータを流し込んで、件数分のファイルを一括生成します。

```
data.csv（100行）＋ template.xlsx  →  document_001.xlsx 〜 document_100.xlsx
```

## 3. Excel集計マクロ

**→ 詳細: [VBA/README.md](VBA/README.md)**

Excelのデータシートを読み込み、項目別に自動集計して別シートへ出力します。

```
【データシート】        【集計シート（自動生成）】
経費A | 5,000          経費A | 7,000
経費B | 3,000    →    経費B | 4,500
経費A | 2,000          経費C | 8,000
経費C | 8,000
```
