# VBA 自動化ツール

## ツール1: Excel集計マクロ

**「毎月Excelで手作業している集計を、ボタン1つで完了させます」**

### 解決する課題

- 項目別の合計を手動でSUMIF関数で計算している
- データが増えるたびに数式の範囲を直し直す
- 集計結果を別シートにコピペする手間

### Before / After

```
【Beforeのデータシート】        【Afterの集計シート（自動生成）】
項目    | 金額                  項目  | 合計
経費A   | 5,000                経費A | 7,000
経費B   | 3,000      →         経費B | 4,500
経費A   | 2,000                経費C | 8,000
経費C   | 8,000
経費B   | 1,500
```

毎月20分かかっていた集計が **1クリック・数秒** で完了。

### 使い方

1. Excelを開く（データは「データ」シートに用意）
2. `Alt + F11` でVBAエディタを起動
3. メニュー「ファイル」→「ファイルのインポート」→ `VBA-001_集計マクロ.bas` を選択
4. VBAエディタを閉じてExcelに戻る
5. `Alt + F8` → `CollectData` を選択 → 「実行」

### カスタマイズ箇所（3行だけ）

```vb
Const INPUT_SHEET  As String = "データ"    ' データが入っているシート名
Const OUTPUT_SHEET As String = "集計"      ' 集計結果を出力するシート名
Const COL_TARGET   As Long = 3             ' 集計のキーにする列番号（A=1, B=2, C=3…）
```

### 動作確認済み環境

- ロジック検証済み（Python同等アルゴリズムで正確性確認）✅
- 経費A:7,000 / 経費B:4,500 / 経費C:8,000（期待値と完全一致）
- 対応Excel：2016以上（Windows）

## ツール2: CSV一括取込マクロ

**「フォルダ内の全CSVをExcelに自動でまとめます」**

### 解決する課題

- 月ごと・担当者ごとに分かれたCSVを手動でコピペしてまとめている
- ファイルが10個以上になると1時間以上かかる

### 使い方

1. `VBA-002_CSV一括取込マクロ.bas` をVBAエディタでインポート
2. スクリプト冒頭の `FOLDER_PATH` を変更
3. `ImportAllCSV` を実行

```vb
Const FOLDER_PATH As String = "C:\work\csv_files\"   ' ← ここだけ変更
```

### 動作結果

フォルダ内の全CSVが1シートに結合されます（2ファイル目以降のヘッダーは自動スキップ）。

## ツール3: 帳票PDF自動出力マクロ

**「請求書・納品書をボタン1つでPDF保存します」**

### 解決する課題

- 毎回「印刷→PDFとして保存→ファイル名をつける」の繰り返し
- ファイル名の付け間違い・保存先の迷い

### 使い方

1. `VBA-003_帳票PDF自動出力.bas` をVBAエディタでインポート
2. 2箇所だけ変更

```vb
Const RANGE_ADDRESS As String = "A1:H40"               ' ← 印刷範囲
Const SAVE_FOLDER   As String = "C:\work\pdf_output\"  ' ← 保存先フォルダ
```

3. `ExportToPDF` を実行 → 「シート名_YYYYMMDD.pdf」で自動保存
