ツール1

CSV整形・クレンジング

「毎回Excelで手直ししているCSVの作業を自動化します」
解決する課題
  • 文字化けするCSV（Shift-JIS → UTF-8変換）
  • セルに混入した余分なスペース
  • 重複行の手動チェック
  • 空行の目視削除
ビフォー／アフター
Before：受け取った時のCSV
1"  株式会社A  "," 田中 ","50000"
2"株式会社B","鈴木花子","30000"
3"  株式会社A  "," 田中 ","50000"
4",","","","  "（空行）
5"株式会社D","山田次郎","20000"
6
7
→
After：整形後のCSV
1株式会社A, 田中太郎, 50000
2株式会社B, 鈴木花子, 30000
3株式会社C, 佐藤一郎, 80000
4株式会社D, 山田次郎, 20000
7行 → 4行（重複2件・空行1件を自動削除）
使い方
コマンド
pip install pandas

python PY-001_CSV整形クレンジング.py
  入力ファイル.csv  出力ファイル.csv
カスタマイズ（3行だけ）
設定箇所
INPUT_ENCODING  = "shift_jis"
OUTPUT_ENCODING = "utf-8-sig"
DROP_DUPLICATES = None
受け取ったCSVの文字コードに合わせる
✓
動作確認済み環境
Python 3.14.3 / pandas 3.0.1　・　入力：Shift-JIS CSV（7行・重複2件・空行1件含む）　・　出力：UTF-8（BOM付き）CSV 4行 ✅

ツール3

Excel集計マクロ（VBA）

「毎月の集計作業をボタン1つで完了させます」
解決する課題
  • 毎月SUMIF関数の範囲を修正している
  • 集計結果を別シートへ手動コピペ
  • データが増えるたびに作業が増える
ビフォー／アフター
Before：データシート
経費A5,000円
経費B3,000円
経費A2,000円
経費C8,000円
経費B1,500円
→
After：集計シート（自動生成）
経費A7,000円
経費B4,500円
経費C8,000円
毎月20分の作業 → 1クリックで完了
インポート手順
手順
1. Alt + F11 → VBAエディタを開く
2. ファイル → インポート
3. VBA-001_集計マクロ.bas を選択
4. Alt + F8 → CollectData → 実行
カスタマイズ（3行だけ）
設定箇所
INPUT_SHEET  = "データ"
OUTPUT_SHEET = "集計"
COL_TARGET   = 3
シート名と集計列番号を変えるだけ
✓
動作確認済み環境
Excel 2016以上（Windows）対応　・　5行データ（3項目・重複あり）→ 正確に集計（経費A:7,000 / B:4,500 / C:8,000）✅
